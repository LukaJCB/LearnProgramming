Tutorial 2 - Data types
In this lesson we'll focus on variables and data types, but wait! Don't leave just yet, it's not actually all that difficult.
We've prepared 3 variable declarations in our code view below. Take a look and hit compile to see, what this program does and then come back.

This might seem a bit confusing, but bear with us. First let's take a look at the basic syntax, the structure of our program. 
println(firstname);
You might remember the println function from last time, so it's time we explained what it actually does. "println" stands for "print line" and as you may have guessed it prints the given value to the console. Also note that every line of code we write ends with a semicolon (;) which denotes an end of expression, so the program knows when to stop evaluating the current expression. Easy so far, right? Let's continue! 
var firstname: string = "Sarah";
Now what you're seeing is what we call a variable declaration. Every variable declaration begins with the keyword 'var' followed by the variable's name.
Then we see the colon symbol (:) followed by the variable's "data type".
What's a data type?
Well let's take a look at the different types found in this program.
  • string - A String is a sequence or a 'string' of character symbols. It can be any length and is denoted by using quotation marks.
  • number - A number is well, a number. A few examples are: -1, 2.4, 12, -234.55621
  • boolean - A boolean value can have exactly two values: true or false. It was named after English mathematician George Boole and is intended to represent the truth values of logic.
