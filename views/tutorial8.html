Tutorial 8 - Functions
In this lesson we're going to talk about functions!
Functions are ways to organize your code and make it reusable. We'll dive right in and have a look at a sample function: 
function getGrade(points: number): string {

    if (points > 94){

        return "A";

    } else if (points > 79){

        return "B";

    } else if (points > 64){

        return "C";

    } else if (points > 49){

        return "D";

    } else {

        return "F";

    }

}

var grade: string = getGrade(74);
println("Your grade is: ");
println(grade);
Does this look crazy to you, yet somewhat familiar? It's the grading system we implemented a few tutorials ago. If we run this code, it's gonna print out "Your grade is : C" to the console.
So we already know what happens inside of that function, but what are those "return"-thingys and what does all this even mean? Well let's start at the beginning.
A function is defined by using the function keyword followed by the function name and then followed by parentheses with what we call "function parameters".
In our case we have one single parameter of type number, but we could have any number of parameters separated by a comma(,). In case of no parameters we would just write nothing into our parentheses.
Here's an example of multiple parameters: 
function add(first: number, second: number): number {
    return first + second;
}
These parameters allow us to pass data into our function. As you can see we can then use this parameter inside our function to calculate things.
After our parameter list you can see a colon (:) followed by another data type. This is the "return type". The return type specifies what data type this function will return when called. In our example we have a return type of string. Meaning that this function will always return a string, which is a grade in our case. With the return keyword we can specify what we want to return. When we return something the rest of the code will be never be executed.
Here's an example: 
function sayHello(): string {
    return "Hello";
    println("This won't get executed :(");
}
