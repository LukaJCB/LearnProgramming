Tutorial 5 - Control flow (continued)
In this lesson we'll continue our focus on control flow.
Now that we know how to use the if-statement let's try out something related. We know how to use the if-statement when the value we passed is true, but what if it isn't? That's where "else"-statement comes in. Let's take a look! 
if (points > 50){

    println("Yay, we passed!");

} else {

    println("Oh no, we failed");

}
We can even stack these if-else patterns like this: 
if (points > 50){

    println("Yay, we passed!");

} else if (points > 0){

    println("Oh no, we failed");

} else {

    println("Oh damn zero points!");

}
In today's exercise you're going to be grading an exam. 50 points is a D, 65 is a C 80 is a B and 95 points will be an A. See if you can create the logic to grade the exam.
