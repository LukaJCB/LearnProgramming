Tutorial 4 - Control flow
In this lesson we'll focus on something called control flow.
Control flow is what happens when we're faced with a decision. Imagine you have test where you need at least 50 points to pass. Now all the tests that have less than 50 points will fail and all the others will obviously pass. This is where our boolean values come in. Let's take a look! 
var has50Points: boolean = true;

if (has50Points){

    println("Yay, we passed!");

}
So what we see here is a classic if-statement. We type "if" followed by a boolean value in parentheses(). Then we follow up with a curly brace, which only gets executed if the above value is true. Let's look at another example. 
if (points < 50){

    println("Oh no, we failed");

}
This time, we again passed a boolean to the if statement. By using the "less than" comparison operator (<) we created a new boolean on the fly! The same thing goes for the "greater than" operator (>), the "is unequal to" operator (!=) and the "is equal to" operator (==). Here, take a look! 
var hasA: boolean = points > 90;

var hasPoints: boolean = points != 0;

var has100Percent: boolean = points == 100;
