Tutorial 6 - Loops
In this lesson we're going to talk about loops!
Loops, as you might have guessed, are a construct that allow you to repeat a certain instruction a given number of times. Let's imagine a scenario where you'd like to print out a certain sentence 10 times. You could do this manually, by writing the statement 10 times, but doing it in a loop is a lot more elegant and flexible. Let's look at the code! 
var counter: number = 0;
while (counter < 10){

    println("I'm counting");

    counter++;

}
Now I hope this doesn't look totally alien to you. At first glance it looks kind of similar to the if-statement right? And it is! In between parentheses we also need to insert a boolean value. This time, however, the code in between the curly braces will keep executing until the boolean value is false. After the loop is done executing we will go back to normal control flow and everything written under the loop will be executed. So let's look at what else we did here. 
counter++;
This line adds one to our counter variable. So after executing it once the value of counter will go from 0 to 1. We can do the opposite thing, subtracting one from our variable by doing this: 
counter--;
So if our counter was 1 before, it will now be 0.

These constructs to increment or decrement a number are SUPER IMPORTANT! Because if you forget to do these things, the boolean value inside the condition of the while loop will NEVER be false. And what does that mean? IT WILL GO ON FOREVER!! Well thankfully thanks to modern computer scientists it's no longer the case. HOWEVER it will still cause a crash, a freeze or other things that are bad. You can try it if you like, though I don't recommend it.
