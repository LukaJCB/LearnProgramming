Tutorial 9 - Classes
In this lesson we're going to talk about classes and objects!
classes and objects play a very important role in modern programming. They're part of a paradigm called Object-Oriented-Programming. This is a very deep field of knowledge so we'll start with the basics. A class is a data type, just like string and boolean and number. We call variables of the type of a class objects. An object can be anything you can think of. Cars can be objects, people can be objects, basically anything you can think of with multiple attributes. Let's start with an example. 
class Person {

    firstName: string;

    lastName: string;

    age: number;

    gender: string;

    hasJob: boolean;

}

var sarah: person = new Person;

sarah.firstName = "Sarah";
sarah.gender = "female";
sarah.age = 18;
sarah.hasJob = false;
Hope this isn't too much to digest. Now let's go through this step by step. First we see the class keyword followed by the class name, which is "Person" in this case. Then we follow up with a curly brace which signals the class body. Inside the class body we define the class's fields or members. These can be of any type you know, including other classes. Once we defined our class we can use it to create objects that are instances of these classes. Also, now with our class definition complete we can use it as a type for our function parameters and function return types. In our example we create a variable named sarah of type person and then assign a new Person to it. The "new"-keyword does exactly that, creating a new object. With your new object you can access all of its fields by using the dot operator (.) and the name of the fields. This way you can group small pieces of data for convenience. Let's take a look at an example of a function that uses classes. 
function isUnderAge(person: Person): boolean {
    if (person.age < 18){
        return true;
    } else {
        return false;
    }
}		

var sarah: Person = new Person;
sarah.age = 18;
println(isUnderAge(sarah));
You can see here quite well that passing in an object is easy and you can use the dot notation for accessing the object in the function. Here's another example where the return type is a class. 
function getOldest(person1: Person, person2: Person): Person {
    if (person1.age > person2.age){
        return person1;
    } else {
        return person2;
    }
}
Here we simply return the oldest of the two persons.
