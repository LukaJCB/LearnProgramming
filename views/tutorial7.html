Tutorial 7 - Loops (continued)
In this lesson we're going to talk about what else you can with loops.
We're gonna jump right in and start with an example of how to determine if a number is odd or even. 
var num: number = 7;
if (num % 2 == 0){

    println("I'm even!");

} else {

    println("I'm odd!");

}
So what are we doing here? What is that weird percentage sign and why are we comparing it to zero?
Okay this isn't all that easy if you're not used to it, so we'll try our best to explain.
The percentage sign(%) stands for the modulo-operator. Modulo operations find the remainder after a division with another number.
In our example we divide 7 by 2 and compare the remainder of that operation to 0.
7 divided by 2 is 3 with a remainder of 1.

So the boolean value will be false and the code in the else-statement will be executed and "I'm odd!" will be printed to the screen. This is true, because every number that can be fully divided by 2 is an even number, so if we were to change the value of our num variable to 8, this would print out "I'm even!".
